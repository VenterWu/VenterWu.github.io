## Hello

This is a personal blog for software notes, experiments, learning logs, and small essays about building useful things.

## What To Expect

Posts here are written to be practical and durable. Some are implementation notes, some are design decisions, and some are short reflections on tools that helped a project move forward.

## Colophon

The site is built with Astro, TypeScript, Tailwind CSS, Markdown content collections, RSS, sitemap generation, dynamic social preview images, and a small amount of vanilla JavaScript.
